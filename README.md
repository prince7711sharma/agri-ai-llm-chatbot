# AgriTech AI Chatbot (LLM)

LLM-based agriculture chatbot using FastAPI and Groq API.

## Features
- English & Hindi support
- Lightweight (Render Free compatible)
- REST API for web integration

## API
POST /api/v1/chat

Request:
{
  "message": "Which soil is best for wheat?",
  "language": "en"
}
